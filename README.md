**E-commerce Data Analysis**
This project focuses on analyzing e-commerce data to extract meaningful insights and trends. The dataset has been explored using data visualization and analytical techniques to understand customer behavior, sales performance, and product trends.

**Key Highlights**
Sales performance analysis
Customer purchasing behavior
Product category insights
Revenue trends over time

**Tools & Technologies**
Power BI
Data Cleaning & Visualization
Dashboard Creation

**Insights**
Identified top-performing products and categories
Analyzed seasonal trends in sales
Found patterns in customer buying behavior

**File Included**
Analysed E-commerce Data.pbix – Power BI dashboard file

**How to Use**
Download the .pbix file
Open it in Power BI Desktop
Explore the interactive dashboard
